# js-secrets
find secrets in js urls

[*] install : 

git clone https://github.com/buggedout-1/js-secrets.git
cd js-secrets
pip install -r requirements.txt

[*] usage   :

python js-secrets.py -l urls.txt 

[*] note    :  

output autosave in secrets .json
